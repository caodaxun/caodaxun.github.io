一般用于无法确定具体类型，不要滥用 dynamic，一般尽量使用 object

object 和 dynamic 区别：object 会在编译阶段检查类型，dynamic 不会

声明常量可以使用 final 或 const，区别：如果常量是编译期就能初始化的用 const，如果常量是运行时期初始化的就用 final
