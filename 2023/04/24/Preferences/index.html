增加系统设置项

PreferenceLoader

开源的基础依赖包，越狱插件的系统设置菜单就是由它提供的，需要越狱商店安装 PreferenceLoader

进入系统设置时，PreferenceLoader 会从 /Library/PreferenceLoader/Preferences 目录（包含子目录）下解析符合规则的 plist 文件，并生成相应的控件动态添加到系统设置中

基本用法

添加一个类似飞行模式的开关效果

  • plist 文件

entry：是配置文件的入口

cell：是一个单元格，PSSwitchCell 表示一个开关控件，更多控件参考 https://iphonedev.wiki/index.php/Preferences_specifier_plist

defaults：关联的用户默认值，实际上是存放于 /var/mobile/Library/Preferences 目录下的一个 plist 文件（会自动创建），文件名就是 defaults 指定的

label：cell 显示的文字信息

icon：cell 前显示的图片，可以使用32 或 29的

PostNotification：设置发生改变后会收到通知

将内容保存到 ANTI.plist ，和 icon.png 放到 iOS 设备 /Library/PreferenceLoader/Preferences 可以建立一个子 CDX 目录, 打开系统设置可以看到

AppList

APPList 用来获取应用程序的信息，允许查看已安装应用程序列表，并获取各种配置信息

AppList 常用键

键 类型 描述
ALNavigationTitle string 导航栏标题
ALSingleEnabledMode bool 仅允许选择单个应用，默认NO
ALSectionDescriptors array 描述各个区域的信息
ALSettingsDefaultValue object 设置默认值
ALSettingsPath string 配置文件路径
ALSettingsKey string key信息，仅ALSingleEnabledModel=true有效
ALSettingsKeyPrefix string key的前缀，仅ALSingleEnabledModel=false有效 默认ALValue-
